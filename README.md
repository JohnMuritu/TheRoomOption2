# Promo code management system

This is a promo code management system

## Installation and Setup Instructions

Clone this repository to your local computer.

### Installation

You will need node and npm installed globally on your machine.
Once you clone the repository, open your favourite terminal and navigate to the root folder of the repository. This is where you should locate the package.json file.
Run the command [npm install]. This will install all the packages required by the system. An internet connection is required for this step.

### Start the system

To start the system, run the command [npm start]

### Access the application

Using your browser, visit [http://localhost:3000]
